# Text Elements

Funkcijska hierarhija (od počasnejše do hitrejše)

1 → lg lg n → lg n → √n → n → n lg n → n² → n³ → n^(lg n) → 2ⁿ → n! → nⁿ

Polinomi ⊏ n^(lg n) ⊏ Eksponenti

## Namig za risanje

- Narisati vodoravno os z markerji funkcij
- Nad vsako napisati primer: sqrt(n), n, n·lg n ...
- Polinome (n, n², n³) obkroži z eno barvo
- n^(lg n) označi z ločeno barvo (ni ne polinom ne eksponent)
- 2ⁿ, n!, nⁿ obkroži z drugo barvo
